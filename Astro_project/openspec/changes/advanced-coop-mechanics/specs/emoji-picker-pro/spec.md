## ADDED Requirements

### Requirement: Categorized Emoji Picker
El selector de emojis DEBE estar organizado por categorías para facilitar la búsqueda rápida durante el juego.

#### Scenario: Switching categories
- **WHEN** el usuario hace clic en el icono de la categoría 'Animales' en el selector
- **THEN** la lista de emojis se filtra para mostrar únicamente animales

#### Scenario: Visual organization
- **WHEN** se abre el selector de emojis
- **THEN** se muestran pestañas o iconos de navegación para las categorías 'Caras', 'Animales', 'Comida' y 'Objetos'
