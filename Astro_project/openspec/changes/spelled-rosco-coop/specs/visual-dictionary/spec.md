## ADDED Requirements

### Requirement: Diccionario de Jeroglíficos Alienígenas
El sistema SHALL permitir la validación y visualización de palabras asociadas a secuencias de 3 emojis (jeroglíficos).

#### Scenario: Visualización por el Traductor
- **WHEN** es el turno de una nueva letra y el jugador tiene el rol 'Traductor'
- **THEN** el sistema muestra la secuencia de 3 emojis correspondiente a la palabra actual en lugar de texto
